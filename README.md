# Connecting-Couples
My CS32 Final Project: 2 Client and Server Mini Games

I want to create a platform where 2 clients can play mini games against each other through a server.

Custom Wordle:
The smart client sends any word.
The dumb client repeatedly guesses and sends their guess to the server which checks the guess against the secret.
